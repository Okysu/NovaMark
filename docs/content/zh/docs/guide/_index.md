# 为创作者

这一部分面向：

- 编剧
- 剧情策划
- 关卡与叙事设计师
- 没有编程背景、但要写互动内容的人

这里不会默认你知道：

- AST 是什么
- 字节码是什么
- 状态机是什么

你在这里最需要理解的是：

- 一段故事怎么写
- 角色和旁白怎么写
- 选择和检定怎么写
- 状态如何影响剧情
- 什么该写进脚本，什么应该交给客户端 UI
